## 🚀 快速开始
```bash
# 1. 克隆
git clone https://github.com/zhenghaochang/xiaomi-vue.git
cd xiaomi-vue

# 2. 启动前端
cd xiaomi-vue
npm install
npm run dev
```
浏览器访问[http://localhost:5173](http://localhost:5173)
